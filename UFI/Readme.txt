UFI: Unreasonably Few Items

By: UFIOES

To install, extract UFI into \Steam\SteamApps\common\TUG\Mods\
Then add "Mods/UFI" to \Steam\SteamApps\common\TUG\Config\mods.txt it MUST be the last mod in the lift.
e.g.

Mods
{
	"Game/Dev"
	"Game/Survival"
	"Game/Creative"
	"Game/Core"
	"Mods/UFI"
}
